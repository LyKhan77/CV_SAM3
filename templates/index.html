How To Prompt SAM

Quick Tips for Better Results

1.
Be Specific: Use clear object names with descriptors
Example: "red safety helmet" instead of "thing"
2.
Add Context: Mention position and surroundings
Example: "worker on the left side"
3.
Use Descriptors: Include color, size, or material
Example: "orange safety vest" or "metal ladder"
4.
One Object Per Prompt: Detect one type at a time
❌ Bad: "helmets and vests" | ✓ Good: "helmet"
5.
Clear Edges Work Best: Objects with defined boundaries
✓ Best: Tools, equipment, people, vehicles
6.
Add Spatial Hints: For multiple similar objects
Example: "person in the center" or "box on top"

Examples:

❌ Bad: "thing" or "stuff"
✓ Good: "safety helmet"
✓✓ Better: "yellow safety helmet on worker"
